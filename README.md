# Smart Task Manager (Python + DSA) 🚀

A command-line task management application built using Python, showcasing Data Structures and Algorithms.

## 🔥 Features
- Add, delete, and view tasks
- Mark tasks as completed
- Search tasks (Linear Search)
- Sort tasks (Priority & Due Date)
- Smart suggestion using Heap (Priority Queue)
- Persistent storage using JSON

## 🧠 DSA Used
- Linear Search
- Sorting
- Heap (Priority Queue)

## 🛠 Tech Stack
- Python

## ▶️ Run Locally

```bash
python main.py
